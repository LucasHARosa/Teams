# criar projeto

```sh
expo init teams
```

# rodar projeto
```sh
npx expo start
```

# Fonte do google
```sh
npx expo install expo-font @expo-google-fonts/roboto
```

# styled-components nessa versão

```sh
npm i styled-components@5.3.9

```
